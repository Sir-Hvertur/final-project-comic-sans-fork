## Kør projektet

1. .env filer:

.env fil i /frontend/eventually/

```
NEXT_PUBLIC_API_URL=http://localhost:4000
```

.env fil i /backend/

```
DB_HOST=****
DB_NAME=****
DB_USER=****
DB_PASSWORD=****
```

2. Installation:

/frontend/eventually/

```sh
npm install
```

3. Kør projektet lokalt:

/backend/

```sh
php -S localhost:4000
```

/frontend/eventually/

```sh
npm run dev
```

/frontend/qr/

```sh
php -S localhost:3001
```
